# Footer 页脚

为整个项目提供页脚信息，自动适应，内容高度不够时，会一直保持在最底部，内容超出则跟随在内容后面。

Footer 组件位于 src/components/Footer 内，如果需要修改页脚信息，可在组件内自定义修改。

# 用法

<script setup lang="ts">
import { Footer } from '@/components/Footer'
</script>

<template>
  <Footer />
</template>
